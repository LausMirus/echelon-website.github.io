# About Echelon Linux

Echelon is a declarative NixOS system configuration repository that defines a complete desktop computing environment for the host machine skynet and user x137. The configuration leverages Nix flakes for reproducible builds and declarative dependency management, creating a Wayland-native desktop environment centered around the niri compositor.
