Echelon Linux : a NixOS multi-user infrastructure

Echelon is a Nix-flake-driven system configuration, integrating NixOS modules, Home Manager and custom development tools including multiple text editors and the Mango window manager flake.It combines overlays for Rust and Niri, packages like Nim and Zen Browser, and editor configurations(Helix, Neovim, Doom Emacs) with shared language servers and formatters.
